> 在 Excel 中正常填写你的表格内容，之后在标题行下插入一行，要居中的填入 :-:，要右对齐的填-:，其它全部填上-。选中表格，复制，然后粘贴到 Markdown 编辑器中。复制出来的 Excel 默认以tab作为单元格分隔符，而 Markdown 则以|为分隔符。顺理成章，当我们使用 | 对原tab分隔符进行替换后，即可生成 Markdown 化的表格。
